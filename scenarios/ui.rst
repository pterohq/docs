The Scenario Page
********************

Once a :ref:`Scenario <scenarios-what>` has been run, the Scenario Page will be how you interact with the results. Let's break down the different parts of the page.


* scenario administration
  
  * renaming
  * colors
  * pinning scenarios
  * refreshing (pro-only)

* behavior sets

* individual behaviors

* feature feedback

  * pinning features
  * deleting features
