Glossary
*****************

.. _glossary:

.. glossary::
  :sorted:

  scenario
    Specified by a :term:`start event` and a :term:`target event`.

  start event
    An :term:`event` which defines the begining of the time window for a :term:`scenario`. Generally this will be a registration or trial start event, but could be anything.

  target event
    An :term:`event` which defines the end of the time window for a :term:`scenario`. Generally this will be a trial conversion or purchase event, but could be anything.
